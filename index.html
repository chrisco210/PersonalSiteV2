    Salt Text

    Salt text uses image steganography to store a string or small file inside of an image, and retrieves it with a key provided to the user. This happens without easily perceptable distortion to the image when encoding smaller files. The program operates mainly through the command line, but a GUI may be added soon. You can find the source on github, or download a compiled jar file from mediafire.

    Arguments
    • -h displays help. Usage: java -jar YourJarPathHere.jar -h
    • -e encodes a string. Usage: java -jar YourJarPath.jar -e "Your string here "
    • -d decodes a string. Usage: java -jar YourJarPath.jar -d
    • -k specifies a key to use. Usage: java -jar You rJarPath.jar -k &qout;Your key here"
    • -i specifies the path of the image to enode in. Usage: java -jar YourJarPath.jar -i "pathGoesHere"
    • -o specifies the path to place the enoded image in. Usage: java -jar YourJarPath.jar -o "outPathGoesHere&qout;
    • -n specifies the entropy (randomness) of an encode. Usage: java -jar YourJarPath.jar -n entropyLevelGoesHere
    • -s specifies the base skip amount with no entropy. Usage: java -jar YourJarPath.jar -s baseSkipAmount
    • -f specifies a file encode. Usage: java -jar YourJarPath.jar -f "fileToEncodeGoesHere"
    • -fd specifies a file decode. java -jar YourJarPath.jar -fd "outPathOfFileGoesHere"
    • -t dumps the key to a text file, or reads code from a text file. Useful for encoding files with extremely long codes. Usage: java -jar YourJarPath.jar -k "PathOfTxtFileGoesHere"
